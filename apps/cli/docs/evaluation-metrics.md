# RAG Test Case 11: Evaluation Metrics

## Goal
Practice measuring RAG quality with useful metrics.

## Scenario
A small benchmark set contains gold passages and expected answers.

## Query
How should we evaluate our RAG system?

## Expected Behavior
- Discusses retrieval metrics like recall@k and MRR
- Includes answer metrics like groundedness and correctness
- Separates retrieval quality from generation quality

## What to Evaluate
- Metric coverage
- Practical usefulness
- Clarity for learners
