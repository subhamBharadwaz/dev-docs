# RAG Test Case 03: Embedding Quality

## Goal
Compare how well embeddings capture semantic similarity.

## Scenario
Two documents use different wording to describe the same concept.

## Query
Why do embeddings matter in retrieval?

## Expected Behavior
- Semantically related content is retrieved even when keywords differ
- The answer explains meaning-based search

## What to Evaluate
- Semantic matching
- Robustness to paraphrasing
- Ranking quality
