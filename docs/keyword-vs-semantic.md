# RAG Test Case 04: Keyword vs Semantic Search

## Goal
Understand when vector retrieval outperforms plain keyword search.

## Scenario
One document uses the term “context injection” while another uses “retrieval augmentation.”

## Query
How is context added to prompts in a RAG pipeline?

## Expected Behavior
- Semantic retrieval finds both related concepts
- The answer connects terminology differences correctly

## What to Evaluate
- Lexical mismatch handling
- Query understanding
- Multi-document synthesis
